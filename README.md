  <h1>Canora Music Player</h1>
</p>
The Aim of this Project is to Provide a Lightweight Fast and Costumizable Music Playback Experience.<br>
<br>

Features:
* Playlists
* Costumization/Themes
* Equalizer
* Shuffle / Repeat
* Notification Bar Widget
* Sorting(By Title/Artist)
* Search
